# Benjamin Franklin OpenClaw Scripts

Scripts created for OpenClaw automation and maintenance.

## Contents
- `tools/` — helper scripts and tooling
- `tools/memcompress/` — modular memory compression engine

## Notable Scripts
- `archive-memory.sh` — archive daily memory logs older than 7 days
- `file-size-monitor.sh` — report files >250 lines
- `memcompress-modular.js` — modular memory compression CLI
- `compress-image.sh` — image compression + metadata
- `check-expired-images.sh` — purge expired images
- `image-stats.sh` — image compression dashboard
- `search-images.sh` — search images by content
- `recent-images-context.sh` — show recent image summaries
- `transcribe-audio.sh` / `transcribe.js` — voice transcription
